# Random Password Generator

## Автор
Сафаргалеев Роман Винерович
## Описание
Графическое приложение для генерации случайных паролей с гибкой настройкой. Позволяет выбирать длину пароля и типы символов (цифры, буквы, спецсимволы). История сгенерированных паролей сохраняется в файл `history.json` и отображается в таблице.

## Требования
- Python 3.8+
- Стандартные библиотеки: `tkinter`, `random` (встроены в Python)

## Установка и запуск
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourname/password-generator.git
   ```
2. Перейдите в папку проекта.
3. Запустите приложение:
   ```
   python main.py
   ```

## Использование
1. Установите желаемую длину пароля с помощью ползунка.
2. Выберите типы символов с помощью чекбоксов.
3. Нажмите кнопку **«Сгенерировать пароль»**.
4. Сгенерированный пароль появится в поле вывода.
5. Пароль автоматически добавится в историю внизу окна.
6. Для очистки истории используйте кнопку **«Очистить историю»**.

## Примеры использования (тесты)
*   **Слабый пароль:** Длина 8, только цифры.
*   **Средний пароль:** Длина 12, буквы (верхний/нижний регистр) + цифры.
*   **Сложный пароль:** Длина 16, все типы символов.
*   **Проверка истории:** Сгенерируйте 3 пароля и убедитесь, что они отображаются в таблице и сохраняются после перезапуска программы.# -4
